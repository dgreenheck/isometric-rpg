# Game Design Doc

## Objective

- Game Type: Isometric RPG
- Art Style: Low-poly Style
- Game Map: Grid-based movement

## Features

- Procedural dungeons
- Turn-based combat
- Enemies/monsters (AI)
- Inventory / Loot
- Weapons (procedural?)
- Quests
- NPCs
- Town
- Character Stats / Classes

## Development Process

- Start simple
- Build layer by layer
- Focus on gameplay first, graphics and polish and optimization come later
